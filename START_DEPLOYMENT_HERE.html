Deploy to Railway

Let's put your backend in the cloud!

✅ ALL FILES READY

📦 What You've Built

You have a complete Node.js backend with authentication, CRUD operations, and dashboard analytics. Right now it only works on your local network. Let's make it work everywhere! 🌍

1

Create Railway Account

Railway is a cloud platform that makes deployment super easy. Sign up using your GitHub account - it's free and takes 2 minutes.

🎫 Go to Railway.app

Click "Login" → "Login with GitHub" → Authorize Railway

2

Push Code to GitHub

Railway deploys from GitHub. If you haven't already, push your backend code to GitHub:

cd taskmanager1/backend git init git add . git commit -m "Ready for deployment" git remote add origin https://github.com/YOUR_USERNAME/dayflow-backend.git git push -u origin main
⚠️ Make sure .env is in .gitignore (never commit secrets!)
3

Switch to PostgreSQL

Railway uses PostgreSQL (not SQLite). Replace your schema file:

cd taskmanager1/backend/prisma cp schema.postgresql.prisma schema.prisma cd .. git add . git commit -m "Switch to PostgreSQL" git push
4

Create Railway Project

In Railway dashboard:

  • Click "+ New Project"
  • Choose "Deploy from GitHub repo"
  • Select your repository
  • Railway starts building (may fail - need database next)
5

Add PostgreSQL Database

In your Railway project:

  • Click "+ New"
  • Choose "Database"
  • Select "PostgreSQL"
  • DATABASE_URL automatically set ✅
6

Set Environment Variables

Click your backend service → "Variables" tab → Add these:

JWT_SECRET=your-32-character-random-string NODE_ENV=production

🔑 Generate JWT Secret

Run this in PowerShell:

-join ((65..90) + (97..122) + (48..57) | Get-Random -Count 32 | ForEach-Object {[char]$_})
7

Deploy & Get URL

Railway deploys automatically! After deployment:

  • Click "Settings" tab
  • Scroll to "Networking"
  • Click "Generate Domain"
  • Copy your URL (e.g., https://your-app.railway.app)
8

Test Deployment

Open TEST_DEPLOYMENT.html in your backend folder:

start backend/TEST_DEPLOYMENT.html

Enter your Railway URL and test all endpoints!

9

Update Mobile App

Open mobile-fixed/src/services/api.js and change:

const API_BASE_URL = 'http://192.168.0.194:3000'; // TO: const API_BASE_URL = 'https://your-app.railway.app';

Save and restart Expo!

10

🎉 Test & Celebrate!

Your app now works worldwide! Test it:

  • Scan QR code with Expo Go
  • Login works
  • All features work
  • Share with friends anywhere!
