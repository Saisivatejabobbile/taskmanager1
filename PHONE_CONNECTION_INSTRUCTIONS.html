📱 Connect Your Phone

Follow these simple steps to see the app on your phone

⚠️ QR Code Not Working?
No problem! Use the manual URL method below instead. It's actually easier!
1
Open Expo Go App

If you don't have it installed:

  • Android: Download from Google Play Store
  • iOS: Download from App Store
2
Enter URL Manually

In Expo Go app, look for "Enter URL manually" or tap the connection icon

3
Copy and Paste This URL:
exp://192.168.0.194:8081
👆 Click to copy, then paste in Expo Go
4
Wait for App to Load

The app will connect and load. You'll see:

  1. Loading screen
  2. DayFlow splash screen
  3. Sign In/Sign Up screen
✅ Connection Successful When You See:
Sign In screen with username and password fields

🌐 Alternative: Test in Web Browser

Can't get the phone connection working? No problem!

Test the backend features in your browser instead:

This is actually the NEW functionality - the backend CRUD operations!
You can register, login, and test creating tasks, events, expenses, and categories.

🔧 Troubleshooting Tips:
  1. Same WiFi: Phone and computer must be on same network
  2. Check Firewall: Windows Firewall might block port 8081
  3. Restart: Close Expo Go completely and try again
  4. Check Terminal: Make sure dev server is still running
