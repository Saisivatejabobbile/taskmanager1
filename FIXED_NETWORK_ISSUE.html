🔧 If It Still Doesn't Work

1. Check WiFi Connection

Make sure both phone and computer are connected to the same WiFi network. Not guest network, not mobile data - same actual WiFi.

2. Restart Expo Go

Close Expo Go app completely on phone, then reopen and scan QR again.

3. Check Firewall

Windows Firewall might be blocking port 3000. Try temporarily disabling it or adding an exception.

4. Test Backend Directly

On your phone's browser, go to: http://192.168.0.194:3000/health
If you see {"status":"ok"}, backend is reachable.
